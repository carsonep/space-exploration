## About The Project

Space Exploration themed website with awesome 3d views of the moon mars and mercury

Features

- 3D maps of the Moon, Mars and Mercury
- Features Story of the day from Nasa regaurding space
- Slight About section of space exploration

### Built With

- [React](https://reactjs.org/)
- [SCSS](https://sass-lang.com/)
- [Arcgis 4.18 API](https://developers.arcgis.com/javascript/latest/)

### Prerequisites

Install latest version of NPM.

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run npm start
   ```sh
   npm run start
   ```
